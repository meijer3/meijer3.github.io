Schiphol Wayfinding API

A public endpoint for indoor routing and PointOfInterests

In this project, is two-fold. On one hand a new version of the python code of the API and dockerization. On the other hand creation of a GeoWeb (geocortex) website for maintaining the Points/Area of interests data. This data can be found in: Schiphol products (like mobile-apps, website), external Airliners (KLM, TUI) and other people. You can work with it aswel! See developer.schiphol.nl

Wayfinding API is one of the few geo-API's. Two more:

Beacon BLE

The beacon API let you enter Minor/Major/RSSI values, and as a result you get a lat-long position. This is not indoorpositioning (Iphone/Android) are way more advanced/better. But for tracking of assets within the terminal. The assets are often connected to a LoRa network

Location Tree

This API is a uniform building/room/zones geolocator.

  • Client: Royal Schiphol Group
