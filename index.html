Glacial Drone Research

Scientific research with a UAV done in Nepal. This reserach is published in Annals of Glaciology: Seasonal surface velocities of a Himalayan glacier derived by automated correlation of unmanned aerial vehicle imagery

The lirung (7234m) with the lirung glacier. It is debris covered (see grey river).

The goal of this research is -still- to check the flow of the Lirung glacier in Nepal. This can be done by flying with a Drone/UAV (unmanned aerial vehicle) a few times a year and record a arial-image. On these images the big boulders and stones of the Lirung can be track and traced. With cross-correlation the movements over time can be found. We reviewed types of correlation models and compare these with the ground-truth. These were done by hand. Further research included finding the best interpolation. This resulted in a human readable map

All of this is done with remote sensing software ENVI. Because this research is continuing for a long time, everything is written in programming code IDL (Interactive Data Language)

  • Date: January 2017
  • Client: Utrecht University

Schiphol Geo API

1 IMDF

Apple designed a standard format for indoor buildings like airports called IMDF. We worked closely with Apple Maps and Safe Software (FME) to build the schiphol IMDF. The conversion from Schiphol data into IMDF is quite complex and ended up in a workspace that exsists of more than 500+ transfomers!

2 Wayfinding API

A public endpoint for indoor routing and PointOfInterests

In this project, is two-fold. On one hand a new version of the python code of the API and dockerization. On the other hand creation of a GeoWeb (geocortex) website for maintaining the Points/Area of interests data. This data can be found in: Schiphol products (like mobile-apps, website), external Airliners (KLM, TUI) and other people. You can work with it aswel! See developer.schiphol.nl

3 Beacon BLE

The beacon API let you enter Minor/Major/RSSI values, and as a result you get a lat-long position. This is not indoorpositioning (Iphone/Android) are way more advanced/better. But for tracking of assets within the terminal. The assets are often connected to a LoRa network

4 Location Tree

This API is a uniform building/room/zones geolocator. There were so many different room definitions on the airport. Ground floor has already 5000+ rooms. The API lets link assets onto a room. They store a UUID. With the API you can get a JSON-repesentative or rendered image of the room, and
